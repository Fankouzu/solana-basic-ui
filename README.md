## 开发

此站点基于 [Vitepress](https://vitepress.dev/zh/) 构建，并部署在 [vercel.com](https://vercel.com)

## 本地调试
如下本地调试，请根据你的运行环境安装 [Node.js](https://nodejs.org)，并进行如下操作：
```
# 克隆本仓库
git clone https://github.com/Fankouzu/solana-basic-ui.git

# 进入仓库根目录
cd solana-basic-ui

# 安装依赖
npm install
```
执行完上述操作，即可进行本地调试：
```
npm run docs:dev
```
